Random Survival Game

An infinite survival game in which you control a stick figure, dodging random obstacles in an abstract 2D platformer setting.

Stick Dungeon

A game that's like a cross between a dungeon crawler and a platformer. Control a stick figure to survive within an infinite randomly generated dungeon.

Hypocube Translocation

A highly challenging 2D abstract puzzle game that involves pushing a square around in order to reach a goal. The game is played in a grid of tiles containing extenders that, when clicked, extend to push (or sometimes pull) objects in a certain direction.

Raycaster

My first attempt at a 3D rendering program.
